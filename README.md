Banwoldang
==========

A helper tool for [gThumb](https://wiki.gnome.org/Apps/Gthumb)

Build
-----

```
go build
```

Run
---

```bash
go run ./main.go  --source-dir=/your/dir  --link-dir=/dir/for/link-tree
```

or

```bash
./banwoldang  --source-dir=/your/dir  --link-dir=/dir/for/link-tree

```

Test
----

```bash
go test
```
